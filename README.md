# Data-Visulization
This is a Data Visualisation Project where I will to do some visualisation on real world problems and try to come up with it's solution

# Data Sources and Citations
The primary dataset used in this project was sourced from Kaggle and originally derived from the UCI Machine Learning Repository.

- **Kaggle Source:** [Heart Disease Dataset](https://www.kaggle.com/datasets/hosammhmdali/heart-disease-dataset) by Hosam Mohamed Ali.
- **Original Source:** Detrano, R., et al. (1989). "International application of a new probability algorithm for the diagnosis of coronary artery disease." American Journal of                               Cardiology.
